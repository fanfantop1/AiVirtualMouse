# AiVirtualMouse - AI 手势虚拟鼠标

基于计算机视觉的虚拟鼠标系统，通过摄像头识别手部姿势来控制鼠标移动、点击和滚轮。

## 功能特性

- **光标移动** — 伸出食指，在空中移动来控制鼠标位置
- **点击操作** — 同时伸出食指和中指，两指靠近即触发点击
- **滚轮控制** — 伸出中指向上滚动，伸出小指向下滚动
- **方向校准** — 支持屏幕坐标旋转校正（按 Q 进入方向调整模式）
- **识别区域调整** — 可缩放和移动绿色识别框（+/-/方向键/WASD）
- **置信度调节** — 按 Z/X 降低/提高手部检测置信度阈值
- **教程面板** — 右侧扩展画布显示手势说明，按 H 或点击按钮开关
- **配置持久化** — 方向偏移、框大小、框位置、置信度自动保存到 config.json

## 环境要求

- Python 3.7+
- 摄像头

## 安装依赖

```bash
pip install opencv-python mediapipe autopy pyautogui numpy pillow
```

## 运行

```bash
python AiVirtualMouse.py
```

## 手势操作

| 手势 | 功能 |
|------|------|
| 伸出食指 | 移动光标 |
| 食指 + 中指 | 点击（两指靠近< 40px） |
| 伸出中指 | 向上滚动 |
| 伸出小指 | 向下滚动 |

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| Q | 进入 / 退出方向调整模式 |
| E | 退出方向调整模式 |
| H | 开关教程面板 |
| Esc | 退出程序 |
| + / - | 放大 / 缩小绿色识别框 |
| 方向键 / WASD | 移动绿色识别框 |
| Z / X | 降低 / 提高置信度阈值 |

## 方向调整模式

按 Q 进入，界面显示方向罗盘，鼠标点击圆盘即可设定方向偏移角度，用于校正摄像头与屏幕的相对方向。

## 项目结构

```
AiVirtualMouse/
├── AiVirtualMouse.py      # 主程序
├── HandTrackingModule.py  # 手部检测模块（基于 MediaPipe）
├── create_icon.py         # 图标生成脚本
└── config.json            # 配置文件（自动生成）
```

## 技术栈

- **OpenCV** — 摄像头采集与图像处理
- **MediaPipe** — 手部关键点检测
- **Autopy** — 鼠标移动与点击
- **PyAutoGUI** — 滚轮控制
- **Pillow** — 中文字体渲染
